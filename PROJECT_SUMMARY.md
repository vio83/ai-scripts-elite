## 🚀 Come Iniziare / Getting Started

### 1. Installazione Rapida
```bash
git clone https://github.com/vio83/ai-scripts-elite.git
cd ai-scripts-elite
./install.sh
```

### 2. Scegli la Versione
```bash

```

- Update interval (CHECK_INTERVAL)
